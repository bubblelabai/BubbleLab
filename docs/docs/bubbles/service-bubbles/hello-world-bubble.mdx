# Hello World Bubble

Simple hello world bubble for testing purposes

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

## Quick Start

```typescript
import { HelloWorldBubble } from '@bubblelab/bubble-core';
const result = await new HelloWorldBubble({
  name: 'Ada',
}).action();
```
